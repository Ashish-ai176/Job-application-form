# Job Application Form

A basic responsive job application form created using HTML and CSS.

## Features

- Personal information section
- Job position and work experience selection
- Education and skills section
- Resume upload field
- Declaration checkbox
- Responsive design for mobile screens

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` in a browser or use the Live Server extension.

## Project Files

- `index.html` - Contains the structure of the job application form.
- `style.css` - Contains the styling and responsive design.
